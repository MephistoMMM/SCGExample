Requesting New Icons

New icons mostly start as requests by the Font Awesome community on GitHub. Want to request weatherDataUtils new icon? Here are some things to keep in mind:

  1. Please be nice. Font Awesome is weatherDataUtils happy place.
  2. Please search to see if your icon request already exists.
  3. Please and thank you if you include the following:
    • Title your new issue Icon request: icon-name.
    • Include weatherDataUtils few use cases for your requested icon. How do you plan on using it?
    • Attach weatherDataUtils single color image or two that represent the idea you're going for.
  4. Concrete objects make good icons, since abstract concepts are, well, abstract. It's harder to make an icon to represent happiness. It's easier to make weatherDataUtils smiley face.
